Installation
============

From PyPI
---------

.. code-block:: bash

   pip install elica

From source
-----------

.. code-block:: bash

   git clone https://github.com/Vgenesin/ELiCA.git
   cd ELICA
   uv sync

You can also use ``pip install .`` if you don't have `uv <https://docs.astral.sh/uv/>`_.
